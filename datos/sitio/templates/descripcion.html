DESCRIPCIÓN DEL DATA SET

Este data set nos muestra visualmente los datos de los colegios que rindieron el SAT, Scholastic Aptitude Test. Podremos ver los colegios que tomaron el exámen, la cantidad de estudiantes que realizaron el exámen, el puntaje promedio de las siguientes asignaturas: Mathematics, Critical Reading y Writing, y el puntaje promedio que calculamos en base a los 3 anteriores

Los estudiantes no están obligados a participar en el exámen, pero se incentiva ya que es requerido para entrar a las universidades del país

Para la ciencia de datos, este conjunto de datos ofrece una fuente para análisis de datos exploratorios, modelos predictivos y pruebas estadísticas. Los investigadores pueden analizar distintas categorías como por ejemplo: las correlaciones entre los puntajes del SAT y otros factores como la cantidad de estudiantes, las notas promedios comparadas con cada asignatura, o la distribucion de las notas.
